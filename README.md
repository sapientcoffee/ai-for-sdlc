## Getting Started: The Basics

Before you order, here are a few one-time setup steps to get you started.

1.  **Login to Google Cloud:**
    `gemini auth login`

2.  **Create the `g` alias:**
    `alias g='gemini'`

3.  **Set your default model:**
    `g config set model gemini-1.5-pro`
